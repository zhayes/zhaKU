    # zhaKU

    基于vue 2.x开发的UI组件库，轻量，简洁。

    图标库基于手机淘宝图标库

    # 安装：

        npm install --save zhaku

    # 引入：

        import zhaku from 'zhaku';
    const {Button, Input} from 'zhaku';

    # 全局引入：

        import Vue from 'vue';
    import zhaku from 'zhaku';
    
    Vue.use(zhaku);

    全局引入后，在模版里使用，只需用小写字母z加组件名，并用-连接。

    例如：<z-button>这个是按钮</z-button>。
